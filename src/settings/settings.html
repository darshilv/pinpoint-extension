Pinpoint Settings

How Site Access Works

Use once on the current page: Click the Pinpoint toolbar button or use the shortcut. Chrome grants temporary access to that tab through your direct action.

Always allow a site: Add the site below and approve Chrome's prompt. Pinpoint will remember that origin and can auto-activate there later.

Allowed Sites

Pinpoint only keeps persistent access to sites you explicitly allow. Adding a site requests Chrome permission for that origin.

Enter exact site origins. Wildcard subdomains are not supported in the UI so approvals stay narrow and review-friendly.

Examples: http://localhost:3000/* · https://app.mycompany.com/*
