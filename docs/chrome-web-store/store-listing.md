# Pinpoint Chrome Web Store Listing Notes

## Positioning

Pinpoint is best framed as a focused webpage review and implementation handoff tool, not a general productivity utility. That keeps the listing aligned with Chrome Web Store single-purpose expectations.

## Recommended short description

Annotate live webpages and turn UI feedback into implementation-ready prompts.

## Recommended detailed description

Pinpoint helps you review a live webpage, click the exact element you want to discuss, and attach a concise note about the change you want made. Use it for design QA, bug review, implementation handoff, and prompt creation for coding agents.

With Pinpoint you can:

- Activate an on-page review mode from the toolbar button or keyboard shortcut.
- Click UI elements to attach feedback directly to the matching surface.
- Review open notes, resolve completed items, and copy all active notes into a clean prompt.
- Configure URL patterns for pages where you want Pinpoint to auto-activate.

## Suggested Chrome Web Store category

`Developer Tools` is the strongest default. `Productivity` is the fallback if you want broader positioning.

## Reviewer talking points

- Single purpose: webpage UI annotation and prompt generation for implementation work.
- Remote code: none.
- Data transmission: none in the current version.
- Site access: Pinpoint uses `activeTab` for one-time access and requests optional origin access only for sites the user explicitly approves.
